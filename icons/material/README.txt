Placez ici les icônes Material Design SVG téléchargées pour le menu :
- dashboard.svg
- group.svg
- speedometer.svg
- settings.svg

Vous pouvez les télécharger depuis https://fonts.google.com/icons ou https://materialdesignicons.com/ ou utiliser les commandes ci-dessous :

Exemple de commandes :
wget -O dashboard.svg https://raw.githubusercontent.com/google/material-design-icons/master/svg/filled/dashboard.svg
wget -O group.svg https://raw.githubusercontent.com/google/material-design-icons/master/svg/filled/group.svg
wget -O speedometer.svg https://raw.githubusercontent.com/Templarian/MaterialDesign-SVG/master/svg/speedometer.svg
wget -O settings.svg https://raw.githubusercontent.com/google/material-design-icons/master/svg/filled/settings.svg

Assurez-vous que le chemin dans le code est correct :
:/icons/material/dashboard.svg
:/icons/material/group.svg
:/icons/material/speedometer.svg
:/icons/material/settings.svg

Si besoin, adaptez le chemin dans dashboardwidget.cpp.
